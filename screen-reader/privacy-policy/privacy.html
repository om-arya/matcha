Privacy Policy

Effective Date: July 3, 2025

Matcha Screen Reader is a Chrome extension designed to improve web accessibility by summarizing data visualizations (such as charts and graphs) found on webpages. We are committed to respecting user privacy and minimizing data collection.

What Information We Collect

The extension does not collect or store any personally identifiable information (PII), user activity, browsing history, or authentication data.

To provide its functionality, the extension accesses image elements on webpages to determine if they are visualizations. These images are analyzed locally in the browser or sent to a local or user-defined API for summary generation. No image data is stored or shared with third parties by the extension.

How We Use the Information

  • To identify and summarize charts or graphs on webpages
  • To read the summaries aloud using Chrome’s built-in Text-to-Speech (TTS) engine

Data Sharing

We do not sell, share, or transfer any user data to third parties.

Third-Party Services

If configured to do so, the extension may send base64-encoded image data to a locally hosted or user-specified Gemini API endpoint for chart summarization. Users control this configuration. By default, no data is sent to external parties.

Security

No data is persistently stored by the extension. All processing is done locally in the browser or via a user-specified API. The extension does not maintain logs, user profiles, or analytics.
